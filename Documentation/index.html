1) Preparation - top

Make sure you have the following:

  1. PHP 5.4+
  2. MySQLi Extension
  3. cURL Extension
  4. GD Library
  5. Mbstring Extension
  6. ZIP Extension
  7. cal_days_in_month Extension
  8. allow_url_fopen

2) Installation - top

The installation is pretty easy, just follow these simple steps and you're good to go:

  1. Upload all files and folders located inside "Script" folder to your server. (Don't upload Script folder, upload what's inside it).
  2. Go to http://www.yoursite.com/install
  3. Installation requirements:
    • Purchase Code: Your SocialKit Purchase Code, you can get it from .
    • MySQL Hostname: Your MySQL host name, e.g: locahost.
    • MySQL Username: Your MySQL username
    • MySQL Password: Your MySQL user password
    • MySQL Database: Your MySQL Database name
    • Site URL: Your Website URL
    • Site Name: Your Website Name
    • Site Title: Your Website Title
    • Site Email: Your Website Email Address
    • Admin Username: Username for your Admin account
    • Admin Password: Password for your Admin account
  4. Once you have written all of the information, click the install button and wait for a while, the installation process may take up to 5 minutes. After the installation is done, you'll get a page that looks like this:
  5. Here's a video tutorial to further guide you if you have any confusion: SocialKit Installation Video Tutorial

3) Frequently Asked Questions (FAQ) - top

  • What PHP framework SocialKit is built on?
    No frameworks have been used. SocialKit is built from scratch using raw php.

  • Which host provider do you recommend for SocialKit?
    We recommend you to use hostgator.com, godaddy.com

  • Can I use this in more than one domain?
    With one purchase license you can use it on one domain name. You need to get new license for each domain name, please check Envato Help Page for more information about licenses http://codecanyon.net/licenses/standard.

  • How to setup social logins?
    For Social Login, you will need to create applications on Facebook / Google+ / Twitter / Instagram. Watch these video tutorials to help you setup social logins:
    • Facebook Login Installation
    • Google Login Installation
    • Twitter Login Installation
    • Instagram Login Installation

  • How to setup Android Webview app?
    To setup Android Webview Application, take a look at this video tutorial.

  • How to setup video chat and audio call feature?
    • Go to https://www.twilio.com/ and register a new account, and buy some balance.
    • Make sure SSL (https) is installed on your domain/server
    • After you signup/login to your account, click here: https://www.twilio.com/console/video/dashboard
    • For "ConfigurationProfileSid", click here: https://www.twilio.com/console/video/profiles
    • For "Live accountSid", click here: https://www.twilio.com/console
    • For "apiKeySid/apiKeySecret", click here: https://www.twilio.com/console/video/dev-tools/api-keys
    • Copy all the keys/SID/Secret and paste them in your Admin Panel > Settings > Audio/Video Calls

  • How to setup Paypal?
    • Login to PayPal, then click here: https://developer.paypal.com/developer/applications/create
    • Set your app name, and email address and click Create App
    • On the top right side, you can see two tabs, SandBox and Live, click on live.
    • Copy the Client ID and Client Secret, and paste them in your Admin Panel > Settings > Payments/Billing.

  • How to setup credit card payments?
    • Login or Create a new account in http://stripe.com
    • Go to account settings and setup your bank account.
    • Now, go to API keys, and copy the live keys.
    • Go to your Admin Panel > Settings > Payments/Billing and paste your Stripe API Keys here.

  • How to add a new language?
    Go to your Admin Panel > Languages. Click on "Add new language" and follow the steps.

  • How to change countries, page categories?
    You can edit them all in your Admin Panel > Languages.

  • Where can I change the texts in welcome page?
    You can change them in your Admin Panel > Languages.

  • I want to modify a text or something about the design...
    You can change everything visually using themes, locate your theme in "themes" folder.

  • Where can I edit the stylesheet?
    You can edit the stylesheet files in "themes/{your_theme}/css".

  • Where can I edit HTML code?
    You can edit html codes in "themes/{your_theme}/layout".

  • Where can I change the logos and background images/videos?
    You can change your logo, favicon, in "themes/{your-theme}/images/logo/". Background videos can be changed in "themes/{your-theme}/wbg-videos" and background images can be changed in "themes/{your-theme}/wbg-images".

  • How can I create a new theme?
    Open "themes" folder. Copy the default theme folder "socialkit". Rename your copied folder, and change the style/html codes/javascript inside it however you want. Activate the theme in your Admin Panel > Themes. Change the theme info in "info.php".

  • I want to change the content of the Privacy Policy, Contact, Disclaimer and/or TOS page.
    Open "themes/theme-name/layout/terms" and open the .phtml file you want to edit, save and upload, as simple as that. :)

  • I have problems/questions related to the script, how can I contact you?
    You can contact me using the SocialKit Support Centre: http://support.socialkit.net/
